0n1f Url shortening service

Server setup

The 0n1f service is running in an Ubuntu machine. It requires nodejs and couchdb.

Run apt-get install couchdb to install couchdb

Install nodejs. Run git clone git://github.com/ry/node.git and then run ./configure && make && make install

Nodejs modules installation

0n1f is dependen on several nodejs modules. Easiest way to install dependecies is to use npm. To install npm use : curl http://npmjs.org/install.sh | sh

    Once you have nmp install install the next modules:
  • nmp install joose joosex-namespace-depended
  • nmp install hash
  • nmp install connect
  • nmp install express
  • nmp install formidable@latest
  • nmp install request
  • npm install bufferlist

REST Interface

  • GET /:id : Look for that id, if id existed then redirect to url.
  • POST /add?url={the url to link to} : Add url to system with an autogenerated unique id. Include the id in reponse (TODO how to return the id)
